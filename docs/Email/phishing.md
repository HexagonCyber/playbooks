# Phishing

Collect the following artefacts from the email message:

```
Sender:  
Sending Server IP:  
Reverse DNS:  
Subject:  
URL: hxxp://  
Domain:  
Attachment Name:  
File MD5 Hash:  
File SHA1 Hash:  
File SHA256 Hash:  
```

---


Email artifact blocking  

```
- Subject line  
- Sending address  
- Sending server IP  
```

---

Web artifact blocking

```  
- URL  
- Domain  
- IP  
```

---

File artifact blocking

```  
- File name  
- File hash
```
